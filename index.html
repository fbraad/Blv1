Easy to use

You don't need to be a technical genius to use Blindvault. Just log in and drag your files into Blindvault, to store them securely.

Secure and private

We use advanced encryption and data scrambling to safeguard your information.

Completely web based

Blindvault is completely web based and works in all new browsers. You don't need to install or configure any software.
